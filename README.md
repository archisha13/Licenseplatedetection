# Licenseplatedetection

**Codes to run license plate detection in image, video and realtime**

```sh
# Run custom yolov4 tensorflow model
python detect.py --weights ./checkpoints/custom-416 --size 416 --model yolov4 --images ./data/images/car.jpg
```
![](https://github.com/archisha13/Licenseplatedetection/blob/main/detection1.png)

```sh
# Run custom yolov4 model on video
python detect_video.py --weights ./checkpoints/yolov4-416 --size 416 --model yolov4 --video 0 --output ./detections/results.avi
```

![](https://github.com/archisha13/Licenseplatedetection/blob/main/video.png)
